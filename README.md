# вљ™пёЏ Installation

Clone the repository

```bash
git clone https://github.com/Chethumalli/Chethumalli-portfolio.git
```

Go to the project

```bash
cd Chethumalli-portfolio
```

Install dependencies

```bash
npm install
```

Create a `.env.local`

```env
RESEND_API_KEY=your_resend_api_key
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```

Production

```bash
npm start
```
